Help & How-To

1: Connect and Go

How to Get Your OpenAI API Key

Contact your organization for an enterprise key, or follow these steps:

  1. Visit the OpenAI API keys page.
  2. Log in to your OpenAI account or create one if you don’t have an account.
  3. Navigate to the API keys section.
  4. Click on "Create new secret key".
  5. Copy the generated key and paste it into the "OpenAI API key" field in the settings.

How to Get Your Claude API Key

Contact your organization for an enterprise key, or follow these steps:

  1. Visit the Claude AI website.
  2. Log in or create an account.
  3. Navigate to the API access or developer section.
  4. Generate a new API key if available.
  5. Copy the key and paste it into the "Claude API key" field in the settings.

How to Get Your Google Docs API Key

Contact your organization for an enterprise key, or follow these steps:

  1. Go to the Google Cloud Console.
  2. Create a new project or select an existing one.
  3. Navigate to "APIs & Services" → "Credentials".
  4. Click "Create Credentials" and choose "API Key".
  5. Restrict the API key to allow access only to Google Docs if needed.
  6. Copy the API key and paste it into the "Google Docs API key" field in the settings.

2: How Does it Work

Request Handovers and Project Reports

Follow these steps to request handovers:

  • Generate a requestor token from the settings page.
  • Share the token with your colleagues or teammates.
  • Receive the curated handover file from the person handling the project and start onboarding.

Create Handovers

Create detailed reports and share your projects effectively:

  • Prepare comprehensive task and project reports.
  • Link offline and online files for easy reference.
  • Create automated file summaries to save time.
  • Securely share encrypted files with your team.

Onboard with Your Handover File

Load your handover file into the app and ask specific questions:

  • Example question: "Give me a summary of the project."
  • Example question: "Where are the details about project X?"
